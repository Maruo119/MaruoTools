# オフライン翻訳ツール - セットアップガイド

## 概要

オフライン翻訳ツールは、インターネット接続なしで日本語と英語の相互翻訳ができるデスクトップアプリケーションです。
Argos Translate の言語モデルを使用し、完全にローカルで翻訳処理を実行します。

## 動作環境

- **OS**: Windows 10/11
- **Python**: 3.7 以上

## セットアップ手順

### 1. Python のインストール確認

まず、お使いのパソコンに Python がインストールされているか確認します。

コマンドプロンプトまたはPowerShellを開いて、以下のコマンドを実行します：

```bash
python --version
```

Python がインストールされていない場合は、[公式サイト](https://www.python.org/downloads/) からダウンロードしてインストールしてください。

### 2. 依存ライブラリのインストール

このアプリケーションを実行するには、`argos-translate` ライブラリが必要です。

コマンドプロンプトまたはPowerShellを開き、以下のコマンドを実行します：

```bash
pip install argos-translate
```

インストール完了まで数分かかる場合があります。

### 3. 初回起動時

アプリケーションを初回起動すると、自動的に日本語⇔英語の翻訳モデルをダウンロードします。

ダウンロード中は以下のメッセージが表示されます：

```
モデルをダウンロード中...
```

**ダウンロード時間の目安**: インターネット接続速度に依存しますが、通常 1～2 分程度かかります。
ダウンロードが完了するまでお待ちください。

## 実行方法

### 方法1: run.bat をダブルクリック（推奨）

1. ファイルエクスプローラーでこのフォルダを開きます
2. `run.bat` ファイルをダブルクリック
3. アプリケーションウィンドウが起動します

### 方法2: コマンドラインから実行

コマンドプロンプトまたはPowerShellを開き、以下のコマンドを実行します：

```bash
cd /path/to/translateByArgostranslate
python translate_gui.py
```

## アプリケーション機能

### 翻訳機能
- **日本語⇒英語**: 日本語テキストを英語に翻訳します
- **英語⇒日本語**: 英語テキストを日本語に翻訳します

### UI/UX機能
- **動的フォントサイズ変更**: スライダーを使用してテキストサイズを 8～20px の範囲で変更可能
- **履歴保持件数の設定**: 翻訳履歴として保持する件数をスピンボックスで 1～100 件の範囲で変更可能
- **履歴の表示と復元**: 過去の翻訳を履歴リストから選択して復元可能
- **履歴のクリア**: 「履歴クリア」ボタンで翻訳履歴を一括削除可能

### コピー機能
- 翻訳結果をワンクリックでクリップボードにコピー可能

## トラブルシューティング

### Q: Python が見つからないエラーが表示される

**A**: 以下を確認してください：

1. Python がインストールされているか確認：
   ```bash
   python --version
   ```

2. インストールされていない場合は、[公式サイト](https://www.python.org/downloads/) からダウンロードしてインストールします

3. インストール時に「Add Python to PATH」にチェックを入れることが重要です

### Q: モデルのダウンロードに失敗した

**A**: 以下を試してください：

1. インターネット接続を確認
2. ファイアウォール設定を確認（必要に応じて許可を与えます）
3. アプリケーションを再起動してダウンロードを再試行

ダウンロード失敗時は、`run.bat` ウィンドウに詳細なエラーメッセージが表示されます。

### Q: 権限エラー（Permission Denied）が表示される

**A**: 以下を試してください：

1. コマンドプロンプトを「管理者として実行」で開き直す
2. アプリケーションを再実行

### Q: アプリケーションが起動しない

**A**: 以下の手順を試してください：

1. コマンドプロンプトを開きアプリケーションフォルダに移動：
   ```bash
   cd /d "C:\path\to\translateByArgostranslate"
   python translate_gui.py
   ```

2. エラーメッセージを確認して原因を特定

3. 依存ライブラリが正しくインストールされているか確認：
   ```bash
   pip list | grep argos-translate
   ```

### Q: 翻訳結果がおかしい

**A**: 翻訳品質は以下の要因に影響されます：

- **言語モデルの特性**: Argos Translate の言語モデルはニューラルベースですが、完全な翻訳を保証するものではありません
- **入力テキストの長さ**: 短いテキストほど翻訳精度が高い傾向があります
- **技術用語**: 専門的な技術用語は正確に翻訳されない場合があります

翻訳品質を改善したい場合は、翻訳テキストを分割して実行してみてください。

